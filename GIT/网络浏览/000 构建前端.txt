安装代码：

npm install -g 名称

brew install 名称

-g 全局-S 格式 -save 附加于打包文件上-D 格式 –save-dev 同上，不过这个是开发者模式本地安装 (将安装包放在 ./node_modules 下)

安装bootstrap3：

npm install bootstrap
npm install -g grunt-cli
cd node_modules/bootstrap
npm install
