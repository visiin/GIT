合并分支的语法：

```c
git merge --no-ff -m "说明" 分支名
Force pushed
```

有得时候必须要用强制发布。

stash暂存空间：

```c
git stash 
git stash list
git stash apply 序列
git stash drop 序列
```

远程信息

```c
git remote -v
git branch --set-upstream branch-name origin/branch-name
git checkout -b branch-name origin/branch-name
```

抓取冲突

```c
git pull
```

标签

```c
git tag
git show <tagname>
git push origin --tags
```
